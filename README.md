# Bot Web Interface

A web dashboard to manage the AI Telegram bot. Built with FastAPI (backend) and React (frontend).

## Features
- View bot logs in real-time.
- Edit environment variables (`.env`).
- Start/Stop/Restart bot via systemd.
- Monitor server resources (CPU, RAM).

## Tech Stack
- **Backend:** FastAPI, Uvicorn, psutil, python-dotenv
- **Frontend:** React, Monaco Editor, Axios, React Hot Toast
- **Infrastructure:** Ubuntu, Nginx, systemd
